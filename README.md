## To Do App

A Simple [To-Do App](https://github.com/lxmwaniky/to_do_app/releases/download/to-do-app/app-release.apk) in Flutter

### Features
- Add a new task
- Mark a task as completed
- Delete a task
- Saves the tasks in local storage
- Responsive UI
- Animations
